# Changelog

All notable changes to Project AETHER will be documented in this file.

## [1.0.0] - 2026-07-11
### Added
- Complete JIT compiler with 260+ keyword support.
- Quantum, multiverse, swarm, and BCI primitives.
- Package manager integration (`aether install`).
- Auto-updating mechanism (`aether self-update`).
- Initial standard library suite (collections, io, net, crypto, math, time, system).
- Cross-platform release builders.
