# AETHER Logo & Brand Assets

## Logo Files

### Main Logo
- `aether-logo.svg` - Vector logo (recommended for web/print)
- `aether-logo.png` - 512x512 PNG
- `aether-logo-256.png` - 256x256 PNG
- `aether-logo-128.png` - 128x128 PNG
- `aether-logo-64.png` - 64x64 PNG (favicon)
- `aether-logo-32.png` - 32x32 PNG (small icon)

### File Icon
- `aether-file-icon.svg` - Vector file icon
- `aether-file-icon.png` - 256x320 PNG
- `aether-file-icon-128.png` - 128x160 PNG
- `aether-file-icon-64.png` - 64x80 PNG

## Brand Colors

- **Deep Space Blue:** `#0A1128` (Primary background)
- **Quantum Purple:** `#7B2CBF` (Primary accent)
- **Neon Purple:** `#9D4EDD` (Secondary accent)
- **Neon Cyan:** `#00F5FF` (Highlight)

## Usage Guidelines

### Logo Usage
- Always maintain aspect ratio
- Minimum size: 32x32 pixels
- Use SVG for web/print when possible
- Don't modify colors or add effects

### File Icon
- Used for `.aether` file associations
- Shows on file explorers when registered

## Generate PNG from SVG

```bash
bash scripts/generate-logo-png.sh
```

Requires ImageMagick to be installed.
