# Email Newsletter: AETHER v1.0.0 Open Source Launch

**Subject: Announcing AETHER — The Post-Quantum Programming Language**

Dear Developer,

We are excited to share AETHER, a new open-source programming language built natively in Rust.

AETHER simplifies quantum, multiverse, and swarm computing. Instead of stack frames, it uses the Unified Context Graph (UCG) to manage state.

## Getting Started

Install the toolchain with one command:

**macOS/Linux:**
```bash
curl -fsSL https://raw.githubusercontent.com/devsamikhan/aether/main/scripts/install.sh | bash
```

Explore the documentation and star the repository on GitHub:
https://github.com/devsamikhan/aether
